# How To Play
- The main game runs in ***game.py*** python script.
- ***Explore***, ***Find Loot***, ***Fight***, ***Win***.
- Options ranges from the numerical values ***1-3***.
- To force quit, ***Run away*** or ***Stay in bed***.
- To win you have to ***Win***. Beat the ***Final Boss***.
